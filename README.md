---

QLC Solo Mining Pool, support stratum protocol.

---

## Build and Run
```shell
make clean build
./gqlc-pool --help
```

## Docker
You can build the docker image yourself or download it from docker hub
### Build docker images

```bash
cd docker
./build.sh
```

### Download docker images from docker hub

```bash
docker pull qlcchain/qlc-pool:latest
```
